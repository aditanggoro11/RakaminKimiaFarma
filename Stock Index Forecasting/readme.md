This repository contains the implementation of a Gated Recurrent Unit with Ordinary Differential Equations (GRU-ODE) network to forecast the LQ45 stock index. Designed to handle irregular or high-frequency time-series data, this model leverages a custom continuous-time neural network architecture to predict 5-minute interval closing prices.  This project serves as the core computational model for the thesis analyzing the performance of GRU-ODE on high-frequency Indonesian financial markets.
